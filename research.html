The gap you fill: Train a deep tabular model directly on the 30–60 blood chemistry markers available in a standard clinical panel, with methylation clock scores as the prediction target. Multi-task across 4 clocks simultaneously. Calibrated uncertainty. Graceful missingness. Deployable for $30/sample.

Your novel architecture: BioAge-Transformer

Core design principles:

  1. Feature tokenization (each biomarker becomes a vector, not a raw float)
  2. Biological pathway-aware attention (metabolic markers attend to each other, etc.)
  3. Learned missing value tokens (not imputation — critical for clinical use)
  4. Multi-task heads (one per clock — shared encoder, separate decoders)
  5. Calibrated uncertainty (Gaussian NLL outputs mean + variance per clock)

Architecture in full:

class BioAgeTransformer(nn.Module):
    """
    Multi-task tabular transformer for surrogate epigenetic clock prediction.
    Predicts GrimAge, DunedinPACE, Horvath, and PhenoAge-DNAm
    from standard clinical blood chemistry panels.
    """
    def __init__(
        self,
        n_features: int,          # ~42 blood markers
        d_model: int = 192,
        n_heads: int = 8,
        n_layers: int = 6,
        n_clocks: int = 4,
        n_pathways: int = 6,      # metabolic, lipid, inflammation,
                                   # hormonal, CBC, nutrients
        dropout: float = 0.1,
    ):
        super().__init__()

        # ── Feature tokenization ──────────────────────────────────────
        # Each feature gets a learned identity embedding + value projection
        self.feature_embedding = nn.Embedding(n_features, d_model)
        self.value_projector   = nn.Linear(1, d_model)

        # Learnable missing-value token (one per feature, not global)
        self.missing_tokens = nn.Parameter(torch.randn(n_features, d_model))

        # ── Biological pathway embeddings ────────────────────────────
        # Assign each biomarker to a pathway; pathway embedding added to token
        self.pathway_embedding = nn.Embedding(n_pathways, d_model)
        # feature_to_pathway: [n_features] int tensor — registered as buffer
        # e.g. fasting glucose, insulin, HbA1c → pathway 0 (metabolic)
        #      ApoB, LDL, HDL, TG → pathway 1 (lipid)
        #      hsCRP, homocysteine, fibrinogen → pathway 2 (inflammation)
        #      testosterone, SHBG, TSH, cortisol → pathway 3 (hormonal)
        #      WBC, RBC, MCV, platelets → pathway 4 (CBC)
        #      Vit D, B12, Mg, Zn, omega-3 → pathway 5 (nutrients)

        # ── [CLS] token ───────────────────────────────────────────────
        self.cls_token = nn.Parameter(torch.randn(1, 1, d_model))

        # ── Transformer encoder ───────────────────────────────────────
        encoder_layer = nn.TransformerEncoderLayer(
            d_model       = d_model,
            nhead         = n_heads,
            dim_feedforward = 4 * d_model,
            dropout       = dropout,
            activation    = 'gelu',
            batch_first   = True,
            norm_first    = True,   # Pre-LN — more stable training
        )
        self.transformer = nn.TransformerEncoder(encoder_layer, n_layers)

        # ── Shared projection head ─────────────────────────────────────
        self.shared_head = nn.Sequential(
            nn.LayerNorm(d_model),
            nn.Linear(d_model, 128),
            nn.GELU(),
            nn.Dropout(dropout),
        )

        # ── Per-clock output heads (mean + log_variance) ──────────────
        # Predicting distribution, not point estimate
        self.clock_heads = nn.ModuleList([
            nn.Sequential(
                nn.Linear(128, 64),
                nn.GELU(),
                nn.Linear(64, 2),   # [mean, log_var]
            )
            for _ in range(n_clocks)
        ])

        # ── Auxiliary head: chronological age (regularization) ────────
        self.age_head = nn.Linear(128, 1)

    def tokenize(self, x: Tensor, mask: Tensor) -> Tensor:
        """
        x:    [B, n_features]   — normalized biomarker values
        mask: [B, n_features]   — True where feature is missing
        """
        B, F = x.shape
        feat_ids = torch.arange(F, device=x.device)

        # Identity embedding per feature
        feat_emb = self.feature_embedding(feat_ids)            # [F, D]

        # Value embedding
        val_emb = self.value_projector(x.unsqueeze(-1))        # [B, F, D]

        # Combine: identity + value
        tokens = feat_emb.unsqueeze(0) + val_emb              # [B, F, D]

        # Replace missing features with learned missing token
        if mask is not None:
            missing_emb = self.missing_tokens.unsqueeze(0)    # [1, F, D]
            tokens = torch.where(
                mask.unsqueeze(-1).expand_as(tokens),
                missing_emb.expand(B, -1, -1),
                tokens
            )

        # Add pathway embeddings
        pathway_emb = self.pathway_embedding(self.feature_to_pathway)  # [F, D]
        tokens = tokens + pathway_emb.unsqueeze(0)

        return tokens

    def forward(self, x: Tensor, mask: Tensor = None):
        B = x.shape[0]

        # Tokenize features
        tokens = self.tokenize(x, mask)                       # [B, F, D]

        # Prepend CLS token
        cls = self.cls_token.expand(B, -1, -1)               # [B, 1, D]
        tokens = torch.cat([cls, tokens], dim=1)              # [B, F+1, D]

        # Transformer forward
        out = self.transformer(tokens)                        # [B, F+1, D]
        cls_out = out[:, 0]                                   # [B, D]

        # Shared projection
        shared = self.shared_head(cls_out)                    # [B, 128]

        # Per-clock predictions
        clock_preds = [head(shared) for head in self.clock_heads]
        # Each: [B, 2] → [:, 0] = mean, [:, 1] = log_var

        # Auxiliary age prediction
        age_pred = self.age_head(shared).squeeze(-1)          # [B]

        return clock_preds, age_pred

Training loss:

def gaussian_nll(pred, target):
    """
    Negative log-likelihood of Gaussian.
    pred: [B, 2] — (mean, log_var)
    target: [B] — ground truth clock score
    """
    mean    = pred[:, 0]
    log_var = pred[:, 1].clamp(-6, 6)   # stability
    var     = torch.exp(log_var)
    nll     = 0.5 * (log_var + (target - mean)**2 / var)
    return nll.mean()

def total_loss(clock_preds, clock_targets, age_pred, age_target,
               clock_weights=(1.0, 1.0, 0.8, 0.8),
               lambda_age=0.1):
    """
    clock_preds:   list of [B, 2] — one per clock
    clock_targets: list of [B]   — GrimAge, DunedinPACE, Horvath, PhenoAgeDNAm
    """
    clock_loss = sum(
        w * gaussian_nll(pred, target)
        for w, pred, target in zip(clock_weights, clock_preds, clock_targets)
    )
    age_loss = F.mse_loss(age_pred, age_target)
    return clock_loss + lambda_age * age_loss

Experiment 3: Missing data robustness

Goal: Show clinical deployability — your model degrades gracefully, others don't.

Artificially introduce missingness at rates: 5%, 10%, 20%, 30%, 50%, 70%.

Three missingness mechanisms:

  • MCAR (Missing Completely at Random) — random features missing
  • MAR (Missing at Random) — features more likely missing in older patients, etc.
  • MNAR (Missing Not at Random) — expensive/specialty markers more likely missing

Compare:

  • Your learned missing token approach
  • Mean imputation
  • Iterative imputation (sklearn IterativeImputer)
  • KNN imputation
  • GAIN (generative adversarial imputation)

Expected result: Your learned token outperforms all imputation methods at all missingness levels. Plot as 5-line figure (one per imputation strategy).

Experiment 4: Multi-task vs. single-task

Goal: Show multi-task learning helps, especially for harder clocks.

Train 5 models:

  • Single-task model for GrimAge only
  • Single-task model for DunedinPACE only
  • Single-task model for Horvath only
  • Single-task model for PhenoAge-DNAm only
  • Multi-task model (BioAge-Transformer) predicting all 4

Expected result: Multi-task model outperforms all single-task models, especially on DunedinPACE (pace of aging, hardest to predict from blood alone). The clocks share variance — joint training exploits this.

Experiment 5: Calibration analysis

Section 6 — Novelty Framing (what you claim, precisely)

In the introduction and abstract, these are your 5 distinct novelty claims:

  1. First deep tabular model trained to directly approximate methylation-derived epigenetic clocks from blood chemistry — PhenoAge approximates mortality risk, not GrimAge/DunedinPACE/Horvath. These are different targets. You are the first to bridge this gap at scale.
  2. Multi-task learning across 4 biological age clocks — the clocks measure correlated but distinct aging processes. Joint training exploits shared variance and improves performance, especially on harder clocks like DunedinPACE.
  3. Principled missingness handling via learned tokens — all prior work uses imputation, which leaks distributional assumptions. Your model is trained with random feature masking and handles arbitrary missingness patterns without separate imputation.
  4. Calibrated uncertainty quantification — clinical deployment requires confidence intervals, not point estimates. You provide per-sample prediction intervals that are demonstrably well-calibrated.
  5. Feature minimization analysis establishing minimum effective panels — you establish empirically what the minimum blood panel is for each level of clock prediction performance, enabling cost-aware deployment decisions.
